# Don Agustín Viajes

Aplicación web para Don Agustín Viajes - Agencia de turismo con más de 25 años de experiencia en Uruguay.

## Tecnologías

- React 18 + TypeScript
- Vite
- TailwindCSS
- Supabase (Auth, Database, Storage)
- React Router
- React Hook Form
- Framer Motion

## Características

- Panel administrativo para gestión de viajes, clientes y cotizaciones
- Sistema de autenticación seguro
- Generación de PDFs para cotizaciones
- Blog integrado
- Sistema CRM para seguimiento de clientes
- Reportes y estadísticas

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Configuración

Crear archivo `.env` con las credenciales de Supabase:

```
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
```
